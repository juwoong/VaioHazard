###고장난 문제들

|**액정 문제**|O|X|△|X|X|X|O|X|X|
|**부팅 불량**|O|X|O|O|O|O|X|O|O|

O : 무조건 쓰입니다 (2)
△ : 쓰일 수 있습니다 (1)
x : 안 쓰입니다. (0)

N개의 문제가 발생하면 최대 값은 2*N개. 
즉, **각 요인의 합 / 2N**으로 나누면 거짓말을 해도 걸리지 않을 확률이 됨.
